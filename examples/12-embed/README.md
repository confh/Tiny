# 12 Embed

This example demonstrates the `embedstr` and `embedbin` keywords, which allow you to embed external files directly into your Tiny program at compile time.

- `embedstr` embeds a file as a constant string.
- `embedbin` embeds a file as a constant buffer (binary data).

This is useful for bundling assets, configuration, or other static data within the executable.

```bash
cd examples/12-embed
../../tiny
```
